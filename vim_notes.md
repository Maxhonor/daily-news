## 编辑文件

```
vim /root/.openclaw/openclaw.json
```

## 进入插入模式（可以打字）

按 `i` 键 → 左下角显示 `-- INSERT --`

## 粘贴内容

1. 按 `i` 进入插入模式
2. 鼠标右键 或 `Shift + Insert` 粘贴
3. 按 `Esc` 退出插入模式

## 保存并退出

```
:wq
```

`:wq` = write（保存）+ quit（退出）

## 不保存退出

按 `Esc` → 输入 `:q!` → 回车

`:q!` = quit + 强制（不保存）

## 常用操作

| 命令 | 作用 |
|------|------|
| `i` | 进入插入模式 |
| `Esc` | 退出插入模式 |
| `:w` | 保存 |
| `:q` | 退出 |
| `:wq` | 保存并退出 |
| `:q!` | 不保存强制退出 |
| `dd` | 删除当前行（普通模式下） |
| `u` | 撤销（普通模式下） |
